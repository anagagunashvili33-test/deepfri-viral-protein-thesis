# DeepFRI Viral Protein Thesis

This repository contains code used for DeepFRI-based functional annotation of viral proteins in my master thesis.

## Workflow

1. Set up the DeepFRI Python environment
2. Split large FASTA files into smaller parts
3. Run DeepFRI predictions
4. Merge prediction outputs with viral metadata
5. Generate figures and statistical summaries

data/        Dataset information only
